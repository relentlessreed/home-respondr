# HomeRespondr

HomeRespondr is a polished single-page local-service website centered on fast callback intent, direct contact, and a clean mobile-first presentation. The site keeps the messaging disciplined around the limited confirmed business facts currently available: the business name and phone number.

## Services

- Urgent home issues
- Routine service visits
- Project estimates
- Ongoing property help

## Technology Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Resend for contact-form email delivery

## Local Development

1. Install dependencies.
2. Copy `.env.example` to `.env.local`.
3. Replace the email and site URL values in `.env.local`.
4. Start the development server with `npm run dev`.
5. Open `http://localhost:3000`.
